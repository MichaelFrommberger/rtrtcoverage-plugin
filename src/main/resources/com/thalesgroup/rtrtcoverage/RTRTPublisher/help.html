Ce greffon permet d'obtenir le rapport de couverture issu de RTRT.

Il est possible de spécifier le chemin où récupérer les rapports de couverture. Il est aussi possible de spécifier le chemin où récupérer les fichiers de couverture statiques.

De plus, le tableau permet de définir pour chaque rapport de couverture une indication de réussite et ainsi de rendre prioritaires certaines conditions en forçant le rapport de santé soit vers la réussite (soleil), soit vers l'échec (orage).
